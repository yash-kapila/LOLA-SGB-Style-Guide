Bootstrap

LOLA is built on the Twitter Bootstrap framework v3.0

Default Bootstrap theme is used as much as possible, with some custom css classes contained within a 'macStyles.css' file.

For icons, "Font Awesome Icons" were used.

The custom CSS file should be included below the default Bootstrap CSS files. Normalize.css is used to reset all css rules before loading the Bootstrap CSS files

Bootstrap.min.js is the supporing file for the bootstrap.css for all toggle and script related actions.

Google's JS API is used to draw charts.

mac.js is used for custom, LOLA-specific java script code

tableSorter.js is used to sort tables, etc.

HTML
<script src="js/jquery.min.js"></script>
<script src="js/bootstrap.min.js"></script>
<script src="js/tableSorter.min.js"></script>
<script src="js/mac.js"></script>
<script src="https://www.google.com/jsapi"></script>
<link href="css/normalize.css" rel="stylesheet" />
<link href="css/bootstrap.min.css" rel="stylesheet" />
<link href='css/font-awesome.css' rel='stylesheet' />
<link href="css/macStyles.css" rel="stylesheet" />

Typography

LOLA would follow a standard type pattern for headings and text thus providing a uniform look and feel across the entire application. @mixin feature of SaSS is used to create a type for similar elements which can be included in the CSS classes thus avoiding repetition of CSS code.

Dropdowns

Dropdowns are styled using the Gel Framework as guide with little tweaks specific to LOLA.

Provide default only in case an option needs to be pre-selected in the dropdown. Otherwise, provide title.

Browser View HTML Javascript 

<lola-dropdown 
    title="Select & Add New"
    default="dropdown[0]"
    menu-items="dropdown"
    on-select="dropdownOptionSelected(id)">
</lola-dropdown>
 
vm.dropdown = [
    { id: 1, name: "Add Borrowing Entity" },
    { id: 2, name: "Add Facility" },
    { id: 3, name: "Add Security" },
    { id: 4, name: "Add Guarantee" }
]

vm.dropdownOptionSelected = function(id) {
    console.log(id);
}

Toggle/Radio

Toggle/Radio buttons are styled using the GEL Framework as guide with little tweaks specific to LOLA.

Provide default only in case a button needs to be pre-selected in the toggle list.

Provide a unique name for each radio buttons group.

Browser View HTML Javascript 
<lola-toggle-buttons
    name="conditional-limits"    
    options="toggleOptions"
    default="toggleOptions[0]"
    on-select="toggleOptionsCallback(id)">
</lola-toggle-buttons>
 
vm.toggleOptions = [
    { id: 'ALL', name: 'All' },
    { id: 'YES', name: 'Yes' },
    { id: 'NO', name: 'No' }
];

vm.toggleOptionsCallback = function(id) {
    console.log(id);
}
