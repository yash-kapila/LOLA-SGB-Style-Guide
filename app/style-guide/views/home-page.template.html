Bootstrap

LOLA is built on the Twitter Bootstrap framework v3.0

Default Bootstrap theme is used as much as possible, with some custom css classes contained within a 'macStyles.css' file.

For icons, "Font Awesome Icons" were used.

The custom CSS file should be included below the default Bootstrap CSS files. Normalize.css is used to reset all css rules before loading the Bootstrap CSS files

Bootstrap.min.js is the supporing file for the bootstrap.css for all toggle and script related actions.

Google's JS API is used to draw charts.

mac.js is used for custom, LOLA-specific java script code

tableSorter.js is used to sort tables, etc.

HTML
<script src="js/jquery.min.js"></script>
<script src="js/bootstrap.min.js"></script>
<script src="js/tableSorter.min.js"></script>
<script src="js/mac.js"></script>
